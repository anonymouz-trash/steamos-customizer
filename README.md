# SteamOS Customizer

*SteamOS Customizer* is a project focused on extending the usability of the Steam Deck beyond its default gaming-oriented design. The goal is to simplify the process of transforming the device into a practical laptop replacement while maintaining compatibility with the SteamOS ecosystem.

The project aims to streamline common modifications that many power users perform manually. This includes enabling and re-activating the `pacman` package manager after SteamOS updates, restoring packages that are removed by default, and simplifying the installation of commonly used tools and plugins.

A particular focus lies on integrating and managing popular extensions such as Decky Loader, EmuDeck, and Non-Steam Launchers. By consolidating these steps into a structured and repeatable workflow, SteamOS Customizer reduces the effort required to maintain a customized system across updates.

The long-term objective is to provide a lightweight toolkit that helps users quickly set up a fully functional Steam Deck environment suitable for productivity, development, and general desktop use—effectively bridging the gap between a gaming device and a portable Linux workstation.
